Welcome to this Experiment on "VISUAL PERFORMANCE IN MULTIDIMENSIONAL DATA CHARACTERISATION"

You are invited to take part in a research study to evaluate visual performance in data visualization using different coordinate systems: Cartesian coordinates and Parallel coordinates. This is a study being conducted by Dr. Ulrich Engelke (Research Scientist), Dr. Julian Heinrich (Postdoctoral Fellow), and Dr. Jenny Vuong (Postdoctoral Fellow), CSIRO, Australia.

On this screen, we will provide you with information about the experiment, ask you for your consent should you wish to continue with the experiment, and guide you through a qualification session that you need to pass in order to proceed with the experiment.

Information About the Experiment

  1. What is the purpose of this study?
    Coordinate systems typically represent data points with regard to a set of axes. We aim to identify how well different coordinate systems allow viewers to effectively identify relative distances between such data points. We are particularly interested in the comparison between Cartesian coordinate systems and Parallel coordinate systems. We consider two, three, and four dimensions for each of these coordinate systems.
  2. What does this study involve?
    This study requires you to view 66 coordinate system visualisations. In each coordinate system, you will see three points labelled as A, B, and C. You are asked to judge which of the points B or C is closer to point A. Prior to the actual experiment, we will have a qualification session in which we explain in more detail the task that needs to be performed. The qualification session takes approximately 10 min. You will be allowed to partake in the actual experiment if you pass the qualification session. The whole experiment is estimated to take approximately 30 min.
  3. Are there any possible benefits from participation in this study?
    You will be paid 4.00 USD for this HIT. The experiment provides important information about the effectiveness of multidimensional visualisation techniques to convey data to a user. Any person observing and working with similar data may therefore benefit from more effective and meaningful visualisation.
  4. Are there any possible risks from participating in this study?
    There are no risks associated with this study. You are free to take as much time as needed and you can quit the experiment at any point without having to provide any particular reason.
  5. How will the data collected in this study be used?
    The information that you provide for this research will be treated confidentially and will be used for the following purposes: CSIRO internal storage, statistical analysis and meta analysis with other experiments, research publications in conferences and journals. You will not be identified in any publications resulting from the study except where you have given written permission for this to occur. You may ask for part or all of the information provided to be removed from the study without penalty or explanation.

Visualising Multidimensional Data

As mentioned above, in this study we consider two types of coordinate systems: Cartesian Coordinates and Parallel Coordinates. With both coordinate systems, data points can be visualised using their coordinate-values. The main relevant differences between the two coordinate systems are:
  • in Cartesian coordinate systems axes are drawn perpendicular (90 degree angle) to each other, whereas in Parallel coordinate systems axes are drawn in parallel to each other, and
  • in Cartesian coordinate systems each datum is represented as a point, whereas in Parallel coordinate systems each datum is represented as a line.
For example, consider a datum with two coordinates D = (3, 7). In Cartesian coordinates, we can represent this datum by drawing a point at X1 = 3 and X2 = 7:
Now consider a datum with three coordinates D = (3, 7, 1). To represent this in Cartesian coordinates, we have to use two coordinate systems: one for X1 = 3 and X2 = 7, and another one for X2 = 7 and X3 = 1:
Similarly, we need three Cartesian coordinate systems for data with four coordinates. To represent the same data in parallel coordinates, we can simply add one axis per dimension to the coordinate system and visualise D = (3, 7, 1) with a line that crosses all axes at the respective coordinates at X1 = 3, X2 = 7, and X3 = 1:
