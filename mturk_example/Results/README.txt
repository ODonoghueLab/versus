## Batch_..._results.csv 
contains the answers from workers at mturk. Interesting columns are:

* WorkTimeInSeconds (for all tasks, excl. qualification)
* Answer.X is the answer for id X
* Answer.X_time is the time (in s from January 1, 1970) of completion of that task
* Answer.start is the start time
